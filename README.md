# Smart Study Success Planner

## Overview
An AI-powered study planner built with AWS PartyRock that generates personalized study schedules based on a student's subject, exam date, available study hours, skill level, and study goals.

## Features
- Personalized study schedules
- Weekly milestones
- Key topics to focus on
- Study tips and recommendations
- Motivational messages
- AI-generated planning using natural language prompts

## Technologies Used
- AWS PartyRock
- Generative AI
- Prompt Engineering
- Data Analysis and Visualization

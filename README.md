# Smart Study Success Planner 📚🤖

## Overview
The Smart Study Success Planner is an AI-powered application built using AWS PartyRock. The application generates personalized study plans based on a student's subject, exam date, available study hours, current skill level, and study goals.


## Features
- Personalized study schedules
- Weekly milestones and study timelines
- Key topics to focus on
- Study tips and time management recommendations
- Motivational messages
- Interactive AI-generated responses using natural language prompts

## Technologies Used
- AWS PartyRock
- Generative AI
- Prompt Engineering
- Natural Language Processing (NLP)
